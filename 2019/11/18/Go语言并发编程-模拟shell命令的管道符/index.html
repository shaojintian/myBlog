Go语言并发编程-模拟shell命令的管道符

发表于 2019-11-18

simulate shell command “ps aux | grep QQ”

进程间通信（即IPC问题）是并发中最关键，重要的问题。

基本有三个大的解决方式

1：基于通信

  • 管道pipe
  • 消息队列 message queue

2：基于信号（唯一的异步I/O）

  • 信号 signal

3: 基于同步

  • 信号量 semaphore

现在介绍一种最简单的方式 ： 管道

eg：shell 中 “ |” 就是一个管道符

管道符是一种半双工的通信方式 ， 即一个进程的输出作为另一个进程的输入
